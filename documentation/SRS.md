## <a name="P1">1. Введение</a>
### <a name="P1.1">1.1. Назначение </a>
Приложение предназначено для просмотра и добавлени фотографов.
### <a name="P1.2">1.2. Бизнес-требования</a>
Приложение представляет собой небольшое окно, в котором пользователь может свободно перемещаться по базе данным с фотографами и их портфолио, а также создать свой аккаунт.
### <a name="P1.3">1.3. Аналоги</a>
Аналоги приложения отсутствуют.
